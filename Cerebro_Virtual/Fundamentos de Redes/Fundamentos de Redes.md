# Fundamentos de Redes

Los fundamentos de redes de computadoras abarcan los principios básicos que permiten la comunicación entre dispositivos. Esto incluye la arquitectura de la red, los tipos de redes y los componentes esenciales que hacen posible la interconexión.

## Arquitectura de Red

La arquitectura de una red define cómo se organizan los componentes de la red y cómo se comunican entre sí. Una arquitectura común es el modelo cliente-servidor, pero existen otras como el modelo peer-to-peer.

Una arquitectura simplificada incluye:

*   **Aplicación**: La interfaz con el usuario.
*   **Servicio de comunicaciones**: Proporciona los medios para que las aplicaciones se comuniquen.
*   **Acceso a la red**: La conexión física a la red.

La comunicación en una red se divide en tareas, y cada tarea es realizada por un módulo o capa. Estas capas están organizadas en una pila vertical, y cada capa proporciona servicios a la capa superior.

## Tipos de Redes

Las redes se pueden clasificar por su tamaño y alcance:

*   **[[LAN (Local Area Network)]]**: Una red que se limita a un área geográfica pequeña, como una oficina o un hogar.
*   **[[WAN (Wide Area Network)]]**: Una red que cubre un área geográfica extensa, como un país o un continente.
*   **[[MAN (Metropolitan Area Network)]]**: Una red que abarca un área metropolitana, como una ciudad.
*   **[[PAN (Personal Area Network)]]**: Una red para la comunicación entre dispositivos cercanos a una persona.
*   **[[HAN (Home Area Network)]]**: Una red dentro de un hogar para la comunicación entre dispositivos digitales.

La razón principal para crear una red es **compartir hardware y software**.

## Componentes de una Red

Una red está compuesta por varios elementos:

*   **Fuente**: El dispositivo que origina los datos.
*   **Transmisor**: Convierte los datos en una señal que puede ser transmitida.
*   **Medio de transmisión**: El canal físico a través del cual viaja la señal. Puede ser [[Medios Guiados]] o [[Medios no Guiados]].
*   **Receptor**: Recibe la señal y la convierte de nuevo en datos.
*   **Destino**: El dispositivo al que se envían los datos.

## Desafíos en las Redes

El diseño y la implementación de redes presentan varios desafíos:

*   **Interfaces**: Conectar dispositivos con diferentes características físicas y lógicas.
*   **Representación**: Manejar diferentes formatos de datos.
*   **Sincronización**: Coordinar el tiempo entre el emisor y el receptor.
*   **Gestión de intercambio**: Controlar el flujo de datos.
*   **Detección y corrección de errores**: Manejar los errores que ocurren durante la transmisión.
*   **[[Direccionamiento]] y [[Enrutamiento]]**: Encontrar el camino correcto para los datos.
*   **Seguridad**: Proteger los datos de accesos no autorizados.

Para abordar estos desafíos, se utiliza un enfoque de capas, como se describe en el [[Modelo OSI]] y el [[Modelo TCP/IP]].
